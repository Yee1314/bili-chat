# bili-chat
📺 同屏观影-Google插件


## 1.解压文件
把压缩包解压成一个普通的文件夹。建议把这个文件夹放在电脑里一个不容易被删的地方，因为一旦装好，这个文件夹以后是不能删除也不能移动位置的。
## 2.进入扩展中心
打开电脑上的Chrome浏览器，在顶部地址栏输入 chrome://extensions/并按回车，进入扩展程序管理页面。
## 3.开启开发者模式
在页面的右上角，找到名为”开发者模式”的开关，将其打开(开关会变蓝)
## 4.加载插件
此时页面左上角会出现三个新按钮。点击最左边的”加载已解压的扩展程序”。
## 5.选择文件夹
在弹出的窗口中，找到你刚才解压出来的那个插件文件夹，单击选中它(不要双击点进去)，然后点击右下角的“选择文件夹”。
## 只要操作完这一步，插件列表中就会立刻出现!
